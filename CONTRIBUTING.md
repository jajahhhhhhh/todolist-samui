# Contributing to todolist-samui

Thank you for considering contributing to this project! We welcome all contributions, including bug reports, feature requests, documentation improvements, and code submissions.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Open an issue** to discuss your proposed change if it’s significant.
3. **Make your changes** with clear, descriptive commit messages.
4. **Test your changes** to ensure they work as expected.
5. **Open a Pull Request** and describe your changes.

## Code of Conduct

Please be respectful and considerate in all interactions. Harassment or abusive behavior will not be tolerated.

## Need Help?

If you have questions, open an issue or start a discussion.

---

Everyone is welcome to contribute—no matter your experience level!
